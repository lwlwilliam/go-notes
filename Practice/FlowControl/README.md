### 流程控制

> if else

[IfElse.go](IfElse.go)

无需多讲啦

> switch

[Switch.go](Switch.go)

switch 的判断条件可以为任何数据类型；每个 case 后面跟的是一个完整的程序块，该程序块不需要`{}`，也不需
要 break 结尾，因为每个 case 都是独立的。

> for

[For.go](For.go)

有三种形式：

```go
// 1.
for ...; ...; ... {
    ...
}

// 2. 类似其它语言的 while
for ... {
    ...	
}

// 3. 
for {
    ...	
}
```
