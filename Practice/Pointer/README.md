### 指针

> 定义

指针表示一个变量的地址。

[Definition.go](Definition.go)

有两个符号：

* `&`: 获取一个变量的地址
* `*`: 获取指针所指向的地址的值


> 用途

指针的一大用途就是可以将变量的指针作为实参数传递给函数，从而在函数内部能够直接修改实参所指向的变量值。
